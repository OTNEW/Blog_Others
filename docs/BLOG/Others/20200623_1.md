## 体面的处理

可以登录别人的账号，然后用下面的命令：

```bash
su -p [你自己的用户名]
```

然后就可以在当前用户的情况下登录你的账号，进入自己的home目录后，修改 .bashrc，然后就又可以愉快的玩耍啦。
